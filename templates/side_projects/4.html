As a part of the "100 Days of Code: The complete Python Pro Bootcamp for 2022" course, I worked on the 'portfolio' projects (which had no posted solution, just the idea and general instructions). They are all (except the ones on data science, which can be found on my Google Drive) shared on my Github page.

Here are some of them.

OPAL2bot

Using Selenium module, this bot is used for extracting a parameter from online calculator (PVlighthouse, OPAL2) used for the design of solar cells. Its job is to input custom sets of parameters and extract the calculated values by OPAL.

Space Invaders

Simple, python version of a classic game "Space Invaders". Built using Turtle module, it is a simple (no fancy design for ships) and fun game to play

Data Science - Analysis of Space Missions

This Google Colab notebook has analysis of the missions to space performed by the countries throughout history. Using Pandas and Plotly modules, I analyzed and graphed the data for better understanding and visualisation of trends.

Data Science - Prediciting Earnings

Using the National Longitudinal Survey of Youth 1997-2011 dataset, I performed linear regression model to the data in order to build prediction model which will estimate a person's potential earnings. The Google Colab notebook contains various regression models (using various combinations of feature sets) to see what describes the relationships best.
