Quantifying the performance of a specific material is a necessary step in the development of novel solar cell technologies. Accurate characterization is, however, challenging. Not only does it require good theoretical knowledge of the physical behaviors, but also expensive testing equipment.

This work aims to accurately characterize new materials for their purpose in new solar cell designs. Due to the lack of necessary equipment, here we take an alternative approach. Using simulation software (Silvaco TCAD) together with Python packages (numpy, pandas, scipy) we empirically determine parameters of interest by fitting simulated to experimental data. Python serves as the main driver which controls input parameters of simulations, which are performed in Silvaco. The input parameters are varied until the relative error between experiments and Silvaco is minimized.
